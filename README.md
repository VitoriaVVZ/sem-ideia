# 💭 sem-ideia

Um repositório simples criado para testar, brincar e guardar ideias aleatórias que surgem durante o tédio.

---

## 🚀 Objetivo

Este repositório não tem um propósito fixo. A ideia é justamente essa.

Aqui você ira econtrar:

* Teste de códigos
* Aprender coisas novas (ou nao0
* ideias aleatórias
* projetos pequenos sem compromisso

---

## 📌 Observações

* Nada aqui sera perfeito
* Podera ter varios erros, mas fazem parte do processo
* O importante é se divertir
